You can also compute the \textit{robustness} of a trajectory $\tau$.

\phantom{}

\begin{footnotesize}
\begin{juliaconsole}
using SignalTemporalLogic
τ = [-1.0, -3.2, 2.0, 1.5, 3.0, 0.5, -0.5, -2.0, -4.0, -1.5];
ψ₁ = @formula ◊(sₜ -> sₜ > 0);
ρ₁ = ρ(τ, ψ₁)
ψ₂ = @formula □(sₜ -> sₜ > 0);
ρ₂ = ρ(τ, ψ₂)
\end{juliaconsole}
